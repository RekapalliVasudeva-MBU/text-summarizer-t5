# AI Text Summarizer

A simple local text summarization app using fine-tuned T5 model.

## How to Run

1. Install dependencies:
```bash
pip install -r requirements.txt
```

2. Run the app:
```bash
python app.py
```

3. Open browser at: http://localhost:7860

## Model
- Fine-tuned T5-small on SAMSum dataset
- Hosted on HuggingFace Hub: https://huggingface.co/ValtareVasu/text-summarizer
